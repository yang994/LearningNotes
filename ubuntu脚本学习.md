# 学会偷懒！

## 1
+ `#!`  标记解释用shell  
  `#！/bin/bash` 使用bash作为shell，通用做法
+ 变量操作  
  `readonly 变量名` 设置变量为只读
  `unset 变量名` 删除变量
  `$变量名` 使用变量的方法
+ `echo`命令
  `-e`显示转义字符的选项
+ 计算表达式
  `expr`用于表达式求值  
  例：
  ```
  n=`expr 2 + 2`    #一定要有空格
  echo $n
  ```
  注意 使用乘法运算符 `*` 写作`\*`
+ 关系运算符、布尔运算符、逻辑运算符
  运用在判断语句中  
  表达式写法：
  ```
  [ $a 运算符 $b ]
  ```
